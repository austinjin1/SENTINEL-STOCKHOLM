# Tier A — Real biodiversity figures

Publication figures built **entirely from real public data** — USGS BioData /
EPA Water Quality Portal — with **no synthetic data and no model outputs**.
Every number is reproducible from the Water Quality Portal REST API.

Reproduce with:

```bash
python scripts/build_tierA_figures.py
```

## Data provenance

- **Source:** EPA Water Quality Portal (`waterqualitydata.us`), aggregating USGS
  BioData and STORET biological community surveys.
- **Query:** `characteristicName=Count`, `sampleMedia=Biological`, 2015–2025.
- **Regions (HUC-2):** 02 Mid-Atlantic/Chesapeake, 04 Great Lakes, 05 Ohio River Basin.
- **Scale:** 8,022 unique biological monitoring sites; 401,360 macroinvertebrate/fish
  count records; ~2,300–2,650 taxa per region.

## Figures

| File | Content |
|------|---------|
| `fig1_site_map.png` | Real biomonitoring coverage — all sites, colored by basin. |
| `fig2_richness.png` | Observed macroinvertebrate taxonomic richness (map + per-basin violin). Median ≈ 29 taxa/site. |
| `fig3_bioindicators.png` | EPT (mayfly/stonefly/caddisfly, pollution-sensitive) vs tolerant taxa (worms/midges) occupancy. EPT present at ~73% of sites. |

## Why these matter

These establish that the underlying ecology is real and independently verifiable,
*before* any model is applied. EPT taxa are the real-world counterpart of the
"mayfly" keystone bioindicator used elsewhere in SENTINEL.

Cached intermediate tables (`data/tierA/site_metrics.csv`, `stations_all.csv`) are
committed for convenience; raw `*.parquet` API dumps are git-ignored and
regenerated by the script.
